Réalisations

Inscrube / Agaet

Inscrube / Agaet est un projet de jeu de carte de type TCG que je développe actuellement. Il est entièrement développé sous Godot, prévu pour mobile et PC. J'ai pour but d'implémenter un mode duel en ligne, ainsi qu'un mode solo roguelike.
L'idée du jeu m'est venue en jouant à une version Tabletop Simulator de Incryption en 1 contre 1. Le jeu est pour l'instant assez inspiré de Inscryption, et utilise ses sprites comme place-holder. Je pense à terme les remplacer par des sprites originaux, dans le thème des constellations.
Si vous le souhaitez, les fichers ainsi qu'un wiki est en cours d'écriture sont disponible sur le dépot github du projet pour plus d'infomation sur le jeu.

Not to dash

Not to dash est une démo de jeu de plateforme que j'ai réalisé en 2023 quand je voulais intégrer Piktura (Pole3D à ce moment là).
Le projet est construit sur la base de mon tout premier projet de jeu sur GameMakerStudio2, surtout utilisé à mes débuts pour apprendre le développement de jeux. Le projet devait être un roguelike à génération de terrain procédural, ce qui ne m'était pas trop accessible à ce moment là. J'ai appris énormément sur ce projet, que ce soit au niveau code, mais aussi au niveau game-design et sprite.
La demo a pour but de montrer le fonctionnement d'un moteur simple de jeu de plateforme, sur un niveau introduisant plusieurs mécaniques (pièces à ramasser, pièges, bonus, etc...) au cours du niveau.
Le projet n'a pas de dépot, mais vous pouvez télécharger les fichiers compilés ici.

Bomb It Kid

BombItKid est un petit Bomberman créé à l'occasion de la Nuit du Code avec des amis en 8h. Il est codé en Python avec le framework Pyxel. Ce projet a été pour moi ma première expérience de projet de groupe réessie. Le jeu a été nominé parmit les 3 meilleurs jeux de l'année 2023.
Le script ainsi que les ressources peuvent être lancés via la commande pyxel run ndc.py depuis le dossier où les 2 fichiers se trouvent (Pyxel doit être installé sur la machine)

Back to the Gate

Back to the Gate est une démo d'un jeu 4x et jeu de carte en 3d isométrique développé à l'aide de GameMakerStudio2. Je codais ce jeu à la base comme un projet personnel, mais comme je passait plus de temps sur d'autres projets en même temps, je ne l'ai jamais finit.
A ce moment là mon objectif était d'intégrer Piktura (Pole3D), j'ai donc utilisé cette démo pour illustrer un game concept de mon dossier d'admission, qui parle de développements possibles sur le Lore et le gameplay du jeu.
Vous trouverez ici le lien vers un zip contenant l'exécutable du jeu.
