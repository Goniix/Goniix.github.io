Inscrube / Agaet

Inscrube / Agaet est un projet de jeu de carte de type TCG que je développe actuellement. Il est entièrement développé sous Godot, prévu pour mobile et PC. J'ai pour but d'implémenter un mode duel en ligne, ainsi qu'un mode solo roguelike.
L'idée du jeu m'est venue en jouant à une version Tabletop Simulator de Incryption en 1 contre 1. Le jeu est pour l'instant assez inspiré de Inscryption, et utilise ses sprites comme place-holder. Je pense à terme les remplacer par des sprites originaux, dans le thème des constellations.
Si vous le souhaitez, un wiki est en cours d'écriture existe sur le dépot github du projet pour plus d'infomation sur le déroulement d'une partie.

Not to dash

Not to dash est une démo de jeu de plateforme que j'ai réalisé en 2023 quand je voulais intégrer Piktura (Pole3D à ce moment là).
Le projet est construit sur la base de mon tout premier projet de jeu sur GameMakerStudio2, surtout utilisé à mes débuts pour apprendre le développement de jeux. Le projet devait être un roguelike à génération de terrain procédural, ce qui ne m'était pas trop accessible à ce moment là.
La demo a pour but de montrer le fonctionnement d'un moteur simple de jeu de plateforme, sur un niveau introduisant plusieurs mécaniques (pièces à ramasser, pièges, bonus, etc...) au cours du niveau.
Le projet n'a pas de dépot, mais vous pouvez télécharger les fichiers compilés ici.
